LOGO OFICIAL IPP-UPTAG
======================

El portal detecta automáticamente el logo si lo nombras de cualquiera de
estas formas (en orden de prioridad):

    logoipp.png        ← (el que está en uso)
    logo-ipp.svg
    logo-ipp.png
    logo.svg
    logo.png

Recomendaciones:
- SVG es ideal (nítido a cualquier tamaño y muy liviano). PNG también sirve;
  usa una imagen grande/cuadrada (p. ej. 256x256 o más) para que se vea bien.
- Se muestra dentro de un recuadro blanco tanto en la barra de navegación
  como en el panel de login, así que cualquier versión (color/clara) resalta.

Si no hay ningún archivo, el portal muestra el distintivo de texto "IPP"
como respaldo (no se rompe nada).
